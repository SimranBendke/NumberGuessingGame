# Number Guessing Game Web Application

A Java-based web application that allows users to play a **Number Guessing Game**. This project uses **Servlets, JSP, and the DAO pattern** for managing business logic and database interactions.

## Features

- Play a number guessing game with a system-generated random number.
- Tracks the number of attempts per user session.
- Stores game results in a **database**.
- View and delete game history.
- Maintains **user sessions**.
- Uses **DAO pattern** for clean database operations.

## Technologies Used

- **Java Servlets & JSP** – Web interface and backend logic.
- **JDBC** – Database connectivity.
- **MySQL / any relational database** – Stores game results.
- **HTML, CSS, Bootstrap** – Frontend UI.
- **Apache Tomcat** – Web server.
